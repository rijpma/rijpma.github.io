Openarch deaths

  • Cleaned, deduplicated and (partially) standardized version of Dutch death certificates (c. 1811-1960), retrieved from openarch.nl. The csv files per archive, used to create this dataset, were retrieved in January 2020. Scripts used to create this dataset are available at github/wp4-civreg (version 1.0).
